Uploading/Downloading Data as Excel File

 

 

 

Upload Data as Excel file

Once the template has been edited, it can be uploaded back to the tool by first hitting the "Choose File" button and, after finding the file in your local machine, by then hitting the "Upload" button.

 

·No results of previous analysis will be uploaded even if the spreadsheet contains these results.

·Further analysis of the tool will be restricted to the maximum number of nutrients contained in the uploaded spreadsheet. The tool can always be reset to the default original values by re-loading it.

 

 

Ingredients with "YES" value

In the case of the Upload data as Excel file, only the ingredients with "YES" value are considered as selected, with the exception of the ingredients without a Price. In this case they are automatically unselected during the uploading process. However, predicted values are still calculated for these unselected ingredients.

 

Download Results as spreadsheet

Values in the table can be downloaded to a spreadsheet format by hitting the "Download Results" button. This spreadsheet will have exactly the same headings and values as seen.

 

Results/Outputs as spreadsheet

If the matrix has been solved, results (OUTPUTS) will also be downloaded in the downloaded spreadsheet

 

·The spreadsheet could be used for printout purposes.

·The spreadsheet could be used as template to edit the information and upload back to the tool.
