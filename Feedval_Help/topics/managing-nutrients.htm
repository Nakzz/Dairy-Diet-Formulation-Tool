Managing Nutrients

 

 

 

Nutrients with negative predicted unit costs          

Decide if you want to use nutrients with negative predicted unit costs. Since the tool calculates the nutrients' predicted costs based on simultaneous equations using the market prices of a number of ingredients and their respective compositions, it is likely that some nutrients will solve to have negative costs, so it is up to you to decide if you want to include these (uncheck: default) or include them (check).

 

 

Nutrients Selection

Select the number of nutrients that will be used in a particular analysis (use drop box on the right of "Select nutrients"). The tool allows selecting between 2 and 13 nutrients.

 

 

Combinations of Nutrients

Combination of specific nutrients for an analysis should make sense (e.g., RUP, RDP, and CP could not be selected at the same time) within the ingredients that will be analyzed. The tool will indicate when an error is made and when the calculation is not feasible.

 

 

Blank or Empty nutrients

When a particular nutrient is entered as blank or empty, they are considered as zero during calculations. However, a full column of blank or empty nutrients is not allowed during calculations.

 

 

Number of Nutrients vs Number of Ingredients

The number of selected nutrients has to be smaller or equal than the number of selected ingredients. Otherwise an error message will be displayed.
