Managing Ingredients

 

 

 

Number of Ingredients vs Number of Nutrients

The number of ingredients has to be greater or equal than the number of selected nutrients. Otherwise an error message will be displayed.

 

 

Editing Ingredient Names and their Nutrient Contents

Edit ingredient names (if desired) and their nutrient contents. The tool comes with default nutrient contents, but the user is responsible to assure those represent the conditions of a particular analysis. 

 

 

Editing Ingredient Price Inputs

There user can edit, for every ingredient, the dry matter content (DM%), the price ($/unit), and select the unit in which the ingredient is normally commercialized such as tons (ton), hundred weights (cwt), or kilograms (kg). Notice that when you change the unit, the ingredient price will automatically update.

 

 

Prices for a Number of Ingredients (~26 ingredients)

Prices for a number of ingredients (~26 ingredients) are automatically collected on a daily fashion (daily fetched) from recognized sources. These prices are representative of midwest condition. User is welcome to update or completely change these prices. Note that the tool will "remember" the user inputs within a session of analysis.

 

 

Prices Daily Update (~26 ingredients)

Prices are representative of a particular date (price date on the top right part of the tool). This by default is the most current date (today), but the user is welcome to use the date picker there provided to perform analyses retrospectively with preceding dates.

 

 

Ingredients with "YES" value

In the case of the Upload data as Excel file, only the ingredients with "YES" value are considered as selected, with the exception of the ingredients without a Price. In this case they are automatically unselected during the uploading process. However, predicted values are still calculated for these unselected ingredients. 

 

 

Actual prices as % of Predicted Values

Actual prices as % of predicted value GREATER than 100 (which appear highlighted in red) indicate OVERPRICED ingredients and actual prices as % of predicted value LESSER than 100 (which appear highlighted in green) indicate UNDERPRICED ingredients. Therefore, the lesser this value (last column of the table), the more that ingredient is a BARGAIN. When the ingredient is not part of the solution, but it has a market price the hightligths for "overpriced" and "underpriced" are sligtly different in tone color. Ingredients that do not have a market price will have only their predicted value displayed.
